While you’re learning all of the math symbols and commands, it’s helpful to have a cheatsheet. There are many of these online, google “latex cheat sheet” or “latex math cheat sheet”. Here’s some good ones:

  • http://tug.ctan.org/info/undergradmath/undergradmath.pdf
  • https://www.caam.rice.edu/~heinken/latex/symbols.pdf
  • https://www.nyu.edu/projects/beber/files/Chang_LaTeX_sheet.pdf

There is also the tool detexify where you can draw a math symbol and it will give you the corresponding command.

Common Mistakes

  • The quotation mark character " becomes closing quotes always, so to write opening quotes, use `` (two grave accents (the key to the left of “1” on QWERTY)), so you get ``this is a quote" turn into “this is a quote”.
  • To typeset a backslash, “\”, \\ is line break, the appropriate command is \textbackslash.
  • You don’t need \left and \right for normal height parentheses; it saves some clutter.
